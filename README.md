# EML Explorer

A small React/Vite playground for the `eml(x, y) = exp(x) - ln(y)` primitive.

It includes:
- a live formula explorer
- a manual binary-tree builder
- a symbolic regression demo over noisy generated data

## Local development

```bash
npm install
npm run dev
```

## Tests and build

```bash
npm run lint
npm run test:run
npm run build
```
